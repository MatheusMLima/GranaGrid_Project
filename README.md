# GranaGrid

GranaGrid é um SaaS para organização de gastos pessoais, criado para ajudar pessoas a registrarem e acompanharem seus gastos mensais de forma simples, sem depender de planilhas complexas ou fórmulas de Excel.

A proposta é oferecer uma experiência mais prática, visual e automatizada para quem quer ter controle financeiro no dia a dia.

---

## Sobre o projeto

O GranaGrid permite que o usuário:

- cadastre seus gastos mensais
- organize despesas por categorias
- acompanhe o total gasto no mês
- visualize sua evolução no controle financeiro
- mantenha seus dados salvos de forma segura na nuvem

O objetivo principal é facilitar o controle financeiro pessoal, eliminando a necessidade de criar e manter planilhas manuais.

---

## Motivação

Muitas pessoas acabam desistindo de controlar seus gastos por dependerem de planilhas cheias de fórmulas e configurações.

O GranaGrid surgiu para resolver exatamente esse problema:

> tornar o controle financeiro simples, rápido e acessível para qualquer pessoa.

---

## Tecnologias utilizadas

- Next.js
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- Supabase (Storage para avatares)
- Vercel (deploy)
- Gemini (Estudo e ajudas com Tailwind CSS)
- Figma (Criação do design)

---

## Funcionalidades

- Autenticação de usuários
- Cadastro de despesas
- Organização por categorias
- Resumo mensal
- Armazenamento de avatar do usuário
- Criptografia no banco de dados
